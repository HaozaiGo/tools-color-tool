Color Tool 是什么？

一个用于 HEX/RGB/HSL/HSV/CMYK 互转、调色板生成、颜色对比度检查和 CSS 渐变生成的在线工具。

主要功能

  • Convert HEX, RGB, HSL, HSV, and CMYK
  • Generate harmony-based palettes
  • Create linear and radial CSS gradients
  • Check text contrast against black and white
  • Export CSS variables
  • Explore nearby color variations

适合场景

  • 开发、调试或发布前的日常处理
  • 在浏览器端快速完成任务，减少数据上传
  • 为技术文档、测试数据或线上内容做准备

常见问题

支持哪些颜色格式？

支持 HEX、RGB、HSL、HSV、CMYK 的转换。

可以生成调色板吗？

可以。选择基础颜色和配色方案，即可生成单色、互补、邻近、三等分等调色板。

可以复制 CSS 渐变代码吗？

可以。选择颜色和方向后，可直接复制生成的 CSS gradient。

支持无障碍对比度吗？

支持。工具会计算颜色与黑白文字的对比度，并给出 WCAG 参考评级。

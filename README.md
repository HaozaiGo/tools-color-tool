# 🎨 Color Tool

> 色彩工具 — 颜色格式转换 · 调色板生成 · 渐变色 CSS 生成

## ✨ 功能

| 模块 | 功能 |
|---|---|
| 🔄 **格式转换** | HEX ↔ RGB ↔ HSL ↔ HSV ↔ CMYK 实时互转，任意格式输入即转 |
| 🎨 **颜色预览** | 大色块实时预览 + 系统取色器 + 近似颜色一键切换 |
| 📊 **颜色信息** | 亮度值、色相、饱和度、明度、黑白字对比度、WCAG AA/AAA 无障碍评级 |
| 🎨 **调色板** | 8 种配色方案：单色系、互补色、邻近色、三等分色、四等分色、分裂互补色、暗调、亮调 |
| 📋 **导出调色板** | 一键导出 CSS 变量或下载为 PNG 图片 |
| 🌈 **渐变色** | 线性/径向渐变，多色停点（最多 6 色），实时 CSS 预览，一键复制 |

## 🛠 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas API（调色板图片导出）
- 零外部依赖

## 🚀 部署

支持 Vercel / Netlify / Cloudflare Pages 等静态托管平台。

```bash
npx vercel --prod
```
